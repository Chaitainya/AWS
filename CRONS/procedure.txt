How to run crons:

1. Create a bash file in /home/ubuntu/crons/

eg: calculatecampuscomparativeanalytics1.sh

2. After creating the .sh file paste the below commands

!#/bin/bash
echo "----Start Cron-----"
echo
date
curl -silent https://qa.zomant.com/rest/student/v1/calculatecampuscomparativeanalytics1
echo
echo "-----End cron------"

3. Press --> ctrl + x --> Press y --> and click Enter -- To save

4. Now, go back by cd.. to crons directory and enter the below command,
chmod 755 calculatecampuscomparativeanalytics1.sh (or) chmod 755 /home/ubuntu/crons/calculatecampuscomparativeanalytics1.sh

5. Now, enable cron by below command,
crontab -e

based on the time to call change the stars --> https://crontab.guru/

* * * * * /home/ubuntu/crons/calculatecampuscomparativeanalytics1.sh >> /home/ubuntu/crons/logs/calculatecampuscomparativeanalytics1.log

4. We can get logs under /home/ubuntu/crons/logs/ and ls
